Comments In Javascript:-

Comments basically used for the code read purpose it is easy if someone is reading our code or we are making long projects so it will be easier to undertand the code format.It is not visible on the webpage it is only written in file.

There are two types of comments:-

1.Single line-comment:-

Single line comments can comment the single line in Javascript it doesnt show on the webpage.

Examples of single line comment is:-

// Lorem, ipsum dolor sit amet consectetur adipisicing elit. Est ipsa impedit doloremque? Itaque dolorum corporis atque? Consequuntur
