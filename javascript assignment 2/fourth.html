Variables:-

In javascript variables are used for storing the data the purpose for declaring the variables in javascript is to store the data it is helpful for storing for permanent or temporary data.Variables contains two keyword let and const we can change the variable let after declaring but we cant change the const variable after declaring.

Let Keyword in javascript:-

Let keyword defines we can change the data after using let keyword.Let keyword data will be changed even after declaring.

Example of declaring let keyword:-

let sum=7;
console.log(sum);
sum=10; //reassigned value
console.log(sum);
