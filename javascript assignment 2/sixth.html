Variables in javascript:-

In javascript variables are used for storing the data the purpose for declaring the variables in javascript is to store the data it is helpful for storing for permanent or temporary data.Variables contains two keyword let and const we can change the variable let after declaring but we cant change the const variable after declaring.

Example of declaring let keyword:-

let sum=7;
console.log(sum);
sum=10; //reassigned value
console.log(sum);

Example of declaring const keyword:-

const sum=7;
console.log(sum);

Importance of choosing meaningful variable name

It is important to use meaning variable name because if we are working in a group then it is helpful for code resuability purpose and code readability purpose it is helpful for code reading of others.

Example of a meaningful identifier:-

let age:5
console.log(age)
In an above example it is clearly mentioned that we are storing the age in meaningful identifier where it is clearly defined that we are talking about age.
